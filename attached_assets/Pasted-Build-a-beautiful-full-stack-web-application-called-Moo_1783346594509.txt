Build a beautiful full-stack web application called **MoodBoard AI** using **HTML, CSS, JavaScript, and Node.js (Express)**.

The application should feel premium, calming, and immersive, with a strong focus on animation, smooth transitions, and elegant UI design.

## Theme

Use the **Nature Reset** color palette throughout the application.

```text
Forest Green : #1B4332
Moss Green   : #2D6A4F
Sage         : #95D5B2
Pale Mint    : #D8F3DC
Ivory        : #FAF9F6
```

Create an animated mesh gradient background inspired by forests after rainfall. The colors should slowly blend and drift over time.

Design a modern glassmorphism interface.

* smooth hover effects
* page fade-ins
* premium micro-interactions

```json

```


The finished website should feel like an interactive digital sanctuary.

When someone enters a feeling, the page should transform into a personalized animated world with beautiful visuals, smooth motion, an original poetic response, a matching music vibe, and an elegant color palette. The experience should feel polished, calming, memorable, and portfolio-worthy.
